# AURA Dashboard

A modern React-based interface for interacting with AURA, managing settings, and monitoring system health.

## Tech Stack
- **React 18 & Vite**: Fast development and building.
- **Supabase Client**: Direct interaction with memory services for chat history.
- **TailwindCSS**: Premium, flexible styling.

## Running Locally
```bash
cd dashboard
npm install
npm run dev
```
Dashboard available at: [http://localhost:5173](http://localhost:5173)

## Build
```bash
npm run build
```
The production bundle is served via Nginx in the Docker container.
